MLPM - Multi-Level Progressive Menu

A powerful navigation component for Angular applications

What is MLPM?

MLPM is a standalone Angular component that provides a sophisticated, user-friendly navigation system with support for hierarchical menus, smooth transitions, and customizable theming.

Multi-Level Navigation

Create unlimited menu hierarchies with smooth transitions between levels.

Customizable Themes

Easily adapt the menu's appearance to match your application's design.

Icon Support

Integrated icon component with fallbacks for consistent styling.

Getting Started

npm install ng-mlpm --save

menuItems = {{exampleMenuItems|json}}
